# 自用Gost转发
**快捷命令：gost**

海外VPS：
```
(curl -LfsS https://raw.githubusercontent.com/0xdabiaoge/gost-lite/main/gost.sh -o /usr/local/bin/gost || wget -q https://raw.githubusercontent.com/0xdabiaoge/gost-lite/main/gost.sh -O /usr/local/bin/gost) && chmod +x /usr/local/bin/gost && gost
```
国内VPS：
```
(curl -LfsS https://ghfast.top/https://raw.githubusercontent.com/0xdabiaoge/gost-lite/main/gost.sh -o /usr/local/bin/gost || wget -q https://ghfast.top/https://raw.githubusercontent.com/0xdabiaoge/gost-lite/main/gost.sh -O /usr/local/bin/gost) && chmod +x /usr/local/bin/gost && gost
```
